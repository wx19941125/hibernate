## Welcome to hibernate

 hibernate，数据库持久层框架，ORM框架。

1 对JDBC轻量级封装，全自动化ORM框架。 2 以面向对象的思维操作数据库。

基本思想,将关系型数据库中的数据转化为对象，开发人员可以使用面向对象的思维操作数据库。
